Portfolio

Project : Mesin Pengaduk Minuman Kopi otomatis

  • Merancang dan memprogram PLC Omron untuk mengontrol operasi mesin pengaduk minuman kopi otomatis.
  • Sistem otomatisasi ini menggunakan sensor suhu dan aktuator untuk mendeteksi suhu air dan mengaduk minuman pada kecepatan yang diatur sesuai parameter input.
  • Mengembangkan ladder diagram untuk mengatur siklus otomatisasi, termasuk pengaturan waktu pengadukan, suhu ideal, dan pengoperasian otomatis pengaduk.
  • Mengintegrasikan HMI agar operator dapat mengatur parameter seperti suhu minuman.
